This package uses the official Agibot D1 SDK as a Git submodule needed to build
the D1 Ultra/Edu ROS 2 bridge:

- `third_party/agibot_d1_sdk/include`


The submodule is versioned by Git commit. Review the upstream SDK license before
redistributing binaries or derived packages.
